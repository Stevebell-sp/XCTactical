# XC Tactical

> 為 **Block-Ops**（Minecraft Forge 1.20.1）打造的競技向客戶端模組。
> 作者：**小誠**

鎖定容易被濫用的客戶端設定（碰撞箱顯示、皮膚層），並加入 **Valorant 風格的靜步**：按住一個鍵就能安靜地慢走 —— 比走路慢、比蹲行快，而且**敵人聽不到你的腳步聲**。

---

## ✨ 功能

| 功能 | 說明 |
|---|---|
| **鎖定碰撞箱顯示** | 每 tick 強制關閉實體碰撞箱顯示。玩家就算按 `F3+B` 也會被立刻復原，無法靠顯示碰撞箱取得瞄準優勢。 |
| **鎖定皮膚層** | 每 tick 強制開啟全部皮膚部位（披風、外套、袖子、褲管、帽子層）並同步給伺服器。玩家無法關閉自己的皮膚層，其他人都看得到完整外觀。 |
| **Valorant 靜步**（預設 `Ctrl`） | 按住時：移動速度降為約 55%（**比走路慢、比蹲行快**）、自動取消衝刺，且**腳步無聲**。靠伺服器同步狀態，所以消音對其他玩家也生效。 |

### 關於靜步消音
- 採**雙軌消音**：
  - 原版腳步聲 → 攔 `PlayLevelSoundEvent`。
  - 模組腳步聲 → 攔 Forge 客戶端 `PlaySoundEvent`（`SoundEngine` 層），相容 **[PresenceFootsteps](https://modrinth.com/mod/presence-footsteps)**（它的腳步音效 ID 為 `presencefootsteps:pf_presence.*`、且不經 `Level.playSound`，必須在更底層攔截）。
- 消音判斷以「音效播放位置是否貼近正在靜步的玩家」為準，因此**本地與遠端玩家、原版與模組腳步**都吃得到。
- **走樓梯與攀爬**（梯子 / 藤蔓 / 鷹架）時同樣會套用靜步（消音 + 減速）。
- 目前行為：靜步時**連自己也聽不到**自己的腳步（與瓦羅蘭一致）。

## 📦 安裝

需求：**Minecraft 1.20.1 + Forge 47+**

1. **客戶端**：把 `xctactical-forge-1.20.1-*.jar` 放進 `.minecraft/mods/`。
2. **伺服器**：伺服器端也要放同一個 jar。
   > ⚠️ 靜步消音要對**敵人**生效，必須伺服器端也安裝（負責同步「誰正在靜步」）。若伺服器未安裝，會自動降級為「只有自己聽不到」。

按鍵預設為 **左 Ctrl**，可在遊戲的「選項 → 控制」中自訂。

## 🔧 建置

本專案使用 **Architectury Loom**（forge-only 單一專案）。

```bash
./gradlew build
```

產物在 `build/libs/xctactical-forge-1.20.1-<version>.jar`（忽略 `-sources`、`-dev` 等附屬產物）。

- 編譯目標：Java 17
- 對外不依賴任何函式庫，亦**未使用 Mixin**（全部以 Forge 事件實作）。

---

## 🧩 相容性

- **PresenceFootsteps** ✅ 靜步會一併消掉它取代後的腳步聲。
- 純客戶端功能（鎖碰撞箱 / 鎖皮膚）在任何伺服器皆可用。
- 靜步的「對敵人消音」需伺服器端一併安裝。
